03 — Projects

What I've built

✦ Featured Project

MedScan AI

View on GitHub ↗

MedScan AI came out of my NVIDIA internship capstone — and it's the project I'm most proud of. It's a full-stack application where doctors can register patients, upload medical documents (X-rays, scans, or PDF reports), and instantly receive AI-generated clinical summaries along with patient-friendly explanations. I built the entire backend from scratch using FastAPI and MySQL, implemented JWT-based authentication with role-based access for Admins, Doctors, and Staff, and integrated LLaMA via Groq API to power the AI layer. Every user's data is fully isolated — it's production-ready architecture, not just a demo.
